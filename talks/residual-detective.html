Residual Detective

Anscombe's Quartet & the Datasaurus Dozen — same line, same R², wildly different data.

By the end of this activity you should be able to:
1. Define and compute residuals for a linear regression model and interpret them as prediction errors in context.
2. Construct and interpret residual plots (residuals vs. fitted values) to assess whether linear regression assumptions are reasonable.

What the pattern means

Solved! You've matched every residual pattern to what it tells you about the linear model's assumptions.
Takeaway: Dataset I is the only one where a straight-line model is actually reasonable — its residuals bounce randomly around zero with no pattern. Dataset II's residuals curve, meaning the true relationship isn't linear. Dataset III's residuals are all tiny except one huge outlier that is quietly dragging the whole line toward it. Dataset IV has only one distinct x-value with any spread — that single high-leverage point is entirely determining the slope. Same equation, same R², four completely different stories — you can only tell them apart by plotting the residuals.
